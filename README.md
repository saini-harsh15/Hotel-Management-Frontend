# Hotel Management System - Frontend

A modern React-based frontend for the Hotel Management System. This application provides an intuitive user interface for Customers and Hotel Administrators to manage hotels, bookings, rooms, and reviews through a secure REST API.

> **Backend Repository:**  
> https://github.com/saini-harsh15/Hotel-Management

---

## Features

### Customer

- User Registration & Login
- JWT Authentication
- Browse Hotels
- Search Hotels
- View Hotel Details
- Book Rooms
- View My Bookings
- Submit Reviews

### Hotel Administrator

- Create Hotels
- Edit Hotel Details
- Manage Room Types
- Manage Rooms
- View Hotel Bookings
- Check-In / Check-Out Guests
- View Customer Reviews

## Installation

### Clone the repository

```bash
git clone https://github.com/saini-harsh15/Hotel-Management-System-Frontend.git
```

### Navigate to the project

```bash
cd Hotel-Management-System-Frontend
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run at:

```
http://localhost:5173
```

---

## Backend Configuration

Make sure the Spring Boot backend is running before starting the frontend.

Default backend URL:

```
http://localhost:8081
```

If necessary, update the API URLs inside the service files.
